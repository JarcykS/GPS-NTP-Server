# GPS-NTP-Server
Smal Server NTP on ESP32 and GPS receiver

Celem projektu jest stworzenie małęgo i taniego serwera czasu NTP opartego na module GPS który może pracować w sieciach zamkniętych bez dostępu do interentu, sieciach domowych gdzie jest wymagana synchronizacja wszystkich urządzeń dla jesdnego czasu.
